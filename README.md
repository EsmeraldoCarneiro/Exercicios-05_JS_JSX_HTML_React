# 🚀 Exercícios de ReactJS - Manipulação de Estado (State)

Este repositório contém a quinta rodada de exercícios práticos focados em **Estados (State)** em componentes de classe, lógica de eventos e reatividade no ReactJS.

## 🛠️ Tecnologias e Conceitos

* **State Management**: Gerenciamento de dados internos e mutáveis.
* **Event Handling**: Manipulação de cliques e interações do usuário.
* **Lifecycle (Constructor)**: Inicialização de estados e vinculação de métodos (`bind`).
* **Vite**: Ambiente de desenvolvimento rápido.

---

## 📋 Atividades Realizadas

### 1. Identificação com State Estático

* **Objetivo**: Criar um componente de classe que armazena dados pessoais no objeto `state`.
* **Conceito**: Inicialização do estado no `constructor` e consumo via desestruturação no `render`.
* **Dados**: Nome, Sobrenome e Idade.

### 2. Relógio do Sistema (Objeto Date)

* **Objetivo**: Armazenar um objeto `new Date()` no estado e formatá-lo.
* **Conceito**: Manipulação de objetos globais do JavaScript e formatação de strings de tempo (`toLocaleTimeString`).
* **Resultado**: Exibição da hora atual no formato `00:00:00`.

### 3. Contador Dinâmico (Incremento e Decremento)

* **Objetivo**: Criar um contador interativo que aumenta ou diminui conforme cliques em botões.
* **Conceito**: Uso do método `this.setState()` e vinculação de contexto (`this.bind`).
* **Interação**: Botões "Incrementa" e "Decrementa" atualizando o valor em tempo real.

### 4. Cálculo de Área Reativo

* **Objetivo**: Alterar as dimensões de um retângulo através de um método e exibir a área calculada.
* **Conceito**: Métodos personalizados e recálculo automático de variáveis dentro do `render` após a atualização do estado.
* **Lógica**: Ao mudar a largura de 5 para 6, a área é recalculada instantaneamente.
